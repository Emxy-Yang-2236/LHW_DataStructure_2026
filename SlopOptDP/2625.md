## 题意理解

- $n$ 个工人按 $1, 2, \ldots, n$ 顺序开始工作
- 第 $i$ 个工人开始工作到第 $i+1$ 个工人开始工作间隔 $t[i]$
- 有两次换班，将工人分成三段：$[1, j-1], [j, k-1], [k, n]$（换班发生在工人 $j$ 和工人 $k$ 开始工作的时刻）
- 换班时，所有正在工作的工人立刻下班
- 第三段的工人一直工作到晚班结束

**关键：每个工人的工作时长是什么？**

假设换班在工人 $j$ 和工人 $k$ 开始时发生（$1 < j < k \le n+1$，其中 $n+1$ 代表晚班结束），那么将工人分成三个时段。

对于第一时段中的工人 $i$（$1 \le i < j$）：他从自己开始工作的时刻，一直工作到时刻 $j$ 的换班。工作时长为：
$$T[i] = t[i] + t[i+1] + \cdots + t[j-1] = S[j-1] - S[i-1]$$

其中 $S[i] = \sum_{k=1}^{i} t[k]$，$S[0] = 0$。

对于第二时段中的工人 $i$（$j \le i < k$）：
$$T[i] = S[k-1] - S[i-1]$$

对于第三时段中的工人 $i$（$k \le i \le n$）：
$$T[i] = S[n] - S[i-1]$$

## 目标

最小化：
$$\text{ans} = \min_{1 < j < k \le n+1} \left\{ \sum_{i=1}^{j-1} w[i](S[j-1]-S[i-1]) + \sum_{i=j}^{k-1} w[i](S[k-1]-S[i-1]) + \sum_{i=k}^{n} w[i](S[n]-S[i-1]) \right\}$$

第三段是固定的（晚班结束不可改变，但 $k$ 可变，所以第三段也依赖 $k$）。

## 化简

定义：
- $S[i] = \sum_{m=1}^{i} t[m]$（时间前缀和）
- $W[i] = \sum_{m=i}^{n} w[m]$（后缀和，从 $i$ 到 $n$ 的效率之和）
- $P[i] = \sum_{m=i}^{n} w[m] \cdot S[m-1]$（后缀加权和）

对于一次"在工人 $j$ 处换班"，从 $j$ 到某个结束点 $e$ 这段工人的贡献是：

$$\sum_{i=j}^{e-1} w[i](S[e-1] - S[i-1]) = S[e-1] \sum_{i=j}^{e-1} w[i] - \sum_{i=j}^{e-1} w[i] S[i-1]$$

$$= S[e-1](W[j] - W[e]) - (P[j] - P[e])$$

所以总价值可以写成关于 $j, k$ 的函数。将三段拆开：

**第一段** $(1 \to j-1)$，结束点 $e_1 = j$：
$$V_1(j) = S[j-1] \cdot W[1] - S[j-1] \cdot W[j] - P[1] + P[j]$$

等等，让我更仔细地用上面的公式：
$$V_1(j) = S[j-1](W[1] - W[j]) - (P[1] - P[j])$$

**第二段** $(j \to k-1)$，结束点 $e_2 = k$：
$$V_2(j,k) = S[k-1](W[j] - W[k]) - (P[j] - P[k])$$

**第三段** $(k \to n)$，结束点为晚班结束，即 $e_3 = n+1$：
$$V_3(k) = S[n](W[k]) - (P[k])$$

注意 $W[n+1] = 0$，$P[n+1] = 0$。

总和：
$$V = V_1(j) + V_2(j,k) + V_3(k)$$

展开：
$$V = S[j-1](W[1]-W[j]) - P[1] + P[j] + S[k-1](W[j]-W[k]) - P[j] + P[k] + S[n] \cdot W[k] - P[k]$$

$$= S[j-1] \cdot W[1] - S[j-1] \cdot W[j] + S[k-1] \cdot W[j] - S[k-1] \cdot W[k] + S[n] \cdot W[k] - P[1]$$

$$= S[j-1] \cdot W[1] + W[j](S[k-1] - S[j-1]) + W[k](S[n] - S[k-1]) - P[1]$$

其中 $S[j-1] \cdot W[1]$ 和 $-P[1]$ 中，$W[1]$ 和 $P[1]$ 是常数。

$$V = W[1] \cdot S[j-1] + W[j] \cdot (S[k-1] - S[j-1]) + W[k] \cdot (S[n] - S[k-1]) - P[1]$$

**令 $\text{const} = -P[1]$（常数项不影响优化），定义：**

$$f(j, k) = W[1] \cdot S[j-1] + W[j] \cdot (S[k-1] - S[j-1]) + W[k] \cdot (S[n] - S[k-1])$$

进一步整理：

$$f(j,k) = (W[1] - W[j]) \cdot S[j-1] + W[j] \cdot S[k-1] + W[k] \cdot (S[n] - S[k-1])$$

## DP思路

### 固定 $k$，关于 $j$ 优化

$$f(j,k) = (W[1] - W[j]) \cdot S[j-1] + W[j] \cdot S[k-1] + W[k] \cdot (S[n] - S[k-1])$$

把只跟 $k$ 有关的部分提出来：

$$f(j,k) = \underbrace{W[k](S[n]-S[k-1])}_{\text{只关于 } k} + \underbrace{W[j] \cdot S[k-1]}_{\text{交叉项}} + \underbrace{(W[1]-W[j]) \cdot S[j-1]}_{\text{只关于 }j}$$

定义：
$$g(j) = (W[1] - W[j]) \cdot S[j-1] = W[1] \cdot S[j-1] - W[j] \cdot S[j-1]$$

则：
$$f(j,k) = g(j) + W[j] \cdot S[k-1] + W[k] \cdot (S[n] - S[k-1])$$

对固定 $k$，要最小化 $f(j,k)$ 就是：

$$\min_j \{ g(j) + W[j] \cdot S[k-1] \}$$

这是经典的 **$\min_j \{ b_j + m_j \cdot x \}$ 形式**（其中 $b_j = g(j)$，$m_j = W[j]$，$x = S[k-1]$），这就是**李超线段树**或者**凸包 trick (CHT)** 的标准形式！

## 斜率优化/凸包 trick 思路

### 步骤一：定义直线

对每个候选的换班点 $j$，定义直线：
$$L_j(x) = W[j] \cdot x + g(j)$$

其中：
- 斜率 = $W[j]$（$W$ 是后缀和，随 $j$ 增大而递减）
- 截距 = $g(j) = (W[1] - W[j]) \cdot S[j-1]$

### 步骤二：枚举 $k$，查询最小值

对每个 $k$（从小到大枚举），查询 $x = S[k-1]$ 处所有直线的最小值：

$$h(k) = \min_{2 \le j \le k-1} L_j(S[k-1]) + W[k] \cdot (S[n] - S[k-1])$$

最终答案：
$$\text{ans} = \min_k h(k) - P[1]$$

### 步骤三：利用单调性

- **$S[k-1]$ 随 $k$ 递增**（查询点单调递增）
- **$W[j]$ 随 $j$ 递增而递减**（斜率单调递减）

斜率单调递减 + 查询点单调递增 → 维护一个**上凸包**，用单调队列即可，时间复杂度 $O(n)$。

### 总结算法

```text
1. 预处理 S[], W[], P[], g[]
2. 维护上凸包（单调队列），存放直线 L_j(x) = W[j]*x + g(j)
3. 从 k = 3 到 n+1 枚举：
   a. 将 j = k-1 对应的直线加入凸包
   b. 查询 x = S[k-1] 处的最小值 val
   c. ans = min(ans, val + W[k]*(S[n]-S[k-1]))
4. 输出 ans - P[1]
```

这就是本题 $O(n)$ 斜率优化/凸包 trick 的完整思路。

## 分离变量

把含 $j$、含 $k$、交叉项分开：

$$V = -P[1] + \underbrace{S[j-1]\cdot W[1] - S[j-1]\cdot W[j+1]}_{\text{只含 } j} + \underbrace{S[n]\cdot W[k+1] - S[k-1]\cdot W[k+1]}_{\text{只含 } k} + \underbrace{S[k-1]\cdot W[j+1]}_{\text{交叉项}}$$

定义：
$$g(j) = S[j-1] \cdot (W[1] - W[j+1]) \quad \text{(只关于 } j\text{)}$$
$$r(k) = W[k+1] \cdot (S[n] - S[k-1]) \quad \text{(只关于 } k\text{)}$$

则：
$$V = g(j) + W[j+1] \cdot S[k-1] + r(k) - P[1]$$

## 转化为直线形式

对固定 $k$，要：

$$\min_{j} \left\{ g(j) + W[j+1] \cdot S[k-1] \right\}$$

把 $S[k-1]$ 看作自变量 $x$，对每个 $j$ 定义一条**直线**：

$$\boxed{L_j(x) = W[j+1] \cdot x + g(j)}$$

- **斜率** $m_j = W[j+1]$
- **截距** $b_j = g(j) = S[j-1](W[1] - W[j+1])$
- **查询点** $x_k = S[k-1]$

问题变为：**在 $x = S[k-1]$ 处，求所有直线 $L_j$ 的最小值。**

## ���调性分析

- $W[j+1]$ 是后缀和，$j$ 增大 → $W[j+1]$ 递减 → **斜率单调递减**
- $S[k-1]$ 是前缀和，$k$ 增大 → $S[k-1]$ 递增 → **查询点单调递增**

## 凸包 Trick

求多条直线在某个 $x$ 处的最小值，就是维护直线集合的**下包络线**。

两条直线 $L_a$ 和 $L_b$（$a < b$，所以 $m_a > m_b$），交点为：

$$x^* = \frac{b_b - b_a}{m_a - m_b} = \frac{g(b) - g(a)}{W[a+1] - W[b+1]}$$

当 $x < x^*$ 时 $L_b$ 更优（斜率小），当 $x > x^*$ 时 $L_a$ 更优（斜率大）。

因为斜率单调递减、查询点单调递增，可以用**单调队列**维护：

### 维护规则

队列中直线按斜率递减排列（即按 $j$ 递增）。

**弹队首**：如果队首两条直线的交点 $\le$ 当前查询点 $x_k$，说明第一条直线已经不可能最优，弹出。

**弹队尾**：加入新直线时，如果队尾三条直线中，中间那条永远不会成为最优（被前后两条"夹掉"），弹出中间那条。判断方法就是经典的叉积/交点比较。

### 伪代码

```text
预处理 S[], W[], g[], r[]
const = -P[1]

deque Q  // 存直线编号 j
ans = +∞

for k = 3 to n:
    // 加入直线 j = k-1（保证 j < k）
    new_j = k - 1
    // 弹队尾：检查新直线是否让队尾直线无用
    while |Q| >= 2:
        a = Q[|Q|-2], b = Q[|Q|-1], c = new_j
        // 交点(c,b) <= 交点(b,a) 则弹出 b
        // 即 (b_c - b_b)*(m_a - m_b) <= (b_b - b_a)*(m_b - m_c)
        if cross_check(a, b, c):
            Q.pop_back()
        else:
            break
    Q.push_back(new_j)

    // 查询 x = S[k-1]
    // 弹队首：交点(Q[0], Q[1]) <= S[k-1] 则弹出
    while |Q| >= 2:
        a = Q[0], b = Q[1]
        if L_a(S[k-1]) >= L_b(S[k-1]):
            Q.pop_front()
        else:
            break

    best_j = Q.front()
    val = L_{best_j}(S[k-1]) + r(k)
    ans = min(ans, val)

输出 ans + const
```

### 时间复杂度

每条直线最多入队一次、出队一次，总共 $O(n)$。

## 总结

$$\boxed{V = \underbrace{W[j+1] \cdot x + g(j)}_{\text{直线 } L_j(x)} + r(k) - P[1], \quad x = S[k-1]}$$

| 要素 | 值 | 单调性 |
|------|----|--------|
| 斜率 $m_j$ | $W[j+1]$ | 随 $j$ 递减 |
| 截距 $b_j$ | $S[j-1](W[1]-W[j+1])$ | — |
| 查询点 $x_k$ | $S[k-1]$ | 随 $k$ 递增 |
| 附加项 | $r(k) - P[1]$ | 只关于 $k$ |

斜率递减 + 查询递增 → 单调队列维护下包络线，$O(n)$ 求解。

$$\min_{j} \left\{ g(j) + W[j+1] \cdot S[k-1] \right\}$$

## 斜率优化标准变换

对于固定的 $k$，转移方程是：

$$g(j) + W[j+1] \cdot S[k-1] \le g(j') + W[j'+1] \cdot S[k-1]$$

改写成 $y = kx + b$ 的形式，我们要把**关于 $j$ 的量放到 $y$ 和 $x$，关于 $k$ 的量放到斜率**：

将 $g(j) = -W[j+1] \cdot S[k-1] + \text{val}$ 看成：

$$\boxed{g(j) = (-S[k-1]) \cdot W[j+1] + \text{val}}$$

这就是 $y = kx + b$：

| 符号 | 对应 | 含义 |
|------|------|------|
| $y$ | $g(j) = S[j-1] \cdot (W[1] - W[j+1])$ | 每个决策点 $j$ 对应的纵坐标 |
| $x$ | $W[j+1]$ | 每个决策点 $j$ 对应的横坐标 |
| $k$（斜率） | $-S[k-1]$ | 由当前枚举的 $k$ 决定 |
| $b$（截距） | $\text{val}$，即我们要最小化的值 | $\min_j\{g(j) + W[j+1] \cdot S[k-1]\}$ |

验证：

$$y = kx + b$$
$$g(j) = (-S[k-1]) \cdot W[j+1] + b$$
$$b = g(j) + S[k-1] \cdot W[j+1]$$

这正好就是我们要最小化的那个表达式 ✓

## 几何意义

- 每个决策点 $j$ 对应平面上一个**点** $(W[j+1],\ g(j))$
- 对每个 $k$，用斜率为 $-S[k-1]$ 的直线去"切"这些点，截距最小的就是最优决策
- 最小化截距 → 维护**下凸包**
- $-S[k-1]$ 随 $k$ 增大而递减（斜率递减），所以查询斜率单调递减
- $W[j+1]$ 随 $j$ 增大而递减，所以点的 $x$ 坐标递减 → 加入顺序是**从右往左**

因为点从右往左加入、查询斜率递减，可以用**单调栈/单调队列**从队尾同时加入和查询，或者反过来枚举。具体实现时根据单调性方向选择合适的数据结构即可。
