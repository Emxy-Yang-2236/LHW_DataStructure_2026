好的！这是一个非常关键的问题。很多时候动态规划（DP）给人的印象是“这一行的值只依赖上一行的值”，但在本题中，**同一行的格子互相依赖**，这就导致了必须解方程组。

我们用你提供的具体例子 **$n=3, m=3, x=1, y=2$** 来一步步拆解。

我们的目标是求 $E_{1,2}$（从第一行第二列出发走到第三行的期望步数）。
因为只能往南走（行号增加），不能往北走，所以我们可以**倒着推**：先算出第2行的所有期望值，再算出第1行的所有期望值。

---

### 第一步：确定已知条件（第3行）
题目说走到第 $n$ 行就算结束。
所以第3行的期望步数全是 0：
$$ E_{3,1} = 0, \quad E_{3,2} = 0, \quad E_{3,3} = 0 $$

---

### 第二步：建立第2行的方程组
现在我们站在第2行 $(i=2)$。我们需要求三个未知数：
设 $x_1 = E_{2,1}, \quad x_2 = E_{2,2}, \quad x_3 = E_{2,3}$。

根据题目规则：$E = 1 + \sum P \times E_{next}$。

#### 1. 看位置 (2, 1) —— 最左边
它有3种选择：原地不动、向右走 $(2,2)$、向下走 $(3,1)$。
$$ x_1 = 1 + \frac{1}{3}x_1 + \frac{1}{3}x_2 + \frac{1}{3}E_{3,1} $$
已知 $E_{3,1}=0$，代入整理：
$$ x_1 = 1 + \frac{1}{3}x_1 + \frac{1}{3}x_2 $$
移项（把未知数放到左边）：
$$ \frac{2}{3}x_1 - \frac{1}{3}x_2 = 1 $$
两边同乘3，消去分母：
$$ \mathbf{2x_1 - 1x_2 + 0x_3 = 3} \quad \cdots \text{(方程1)} $$

#### 2. 看位置 (2, 2) —— 中间
它有4种选择：原地、左 $(2,1)$、右 $(2,3)$、下 $(3,2)$。
$$ x_2 = 1 + \frac{1}{4}x_2 + \frac{1}{4}x_1 + \frac{1}{4}x_3 + \frac{1}{4}E_{3,2} $$
已知 $E_{3,2}=0$，代入整理：
$$ x_2 = 1 + \frac{1}{4}x_2 + \frac{1}{4}x_1 + \frac{1}{4}x_3 $$
移项：
$$ -\frac{1}{4}x_1 + \frac{3}{4}x_2 - \frac{1}{4}x_3 = 1 $$
两边同乘4：
$$ \mathbf{-1x_1 + 3x_2 - 1x_3 = 4} \quad \cdots \text{(方程2)} $$

#### 3. 看位置 (2, 3) —— 最右边
它有3种选择：原地、左 $(2,2)$、下 $(3,3)$。
$$ x_3 = 1 + \frac{1}{3}x_3 + \frac{1}{3}x_2 + \frac{1}{3}E_{3,3} $$
已知 $E_{3,3}=0$，代入整理：
$$ x_3 = 1 + \frac{1}{3}x_3 + \frac{1}{3}x_2 $$
移项：
$$ -\frac{1}{3}x_2 + \frac{2}{3}x_3 = 1 $$
两边同乘3：
$$ \mathbf{0x_1 - 1x_2 + 2x_3 = 3} \quad \cdots \text{(方程3)} $$

---

### 第三步：观察矩阵形态（三对角矩阵）

把上面三个加粗的方程写在一起：
$$
\begin{cases}
2x_1 - 1x_2 + 0x_3 = 3 \\
-1x_1 + 3x_2 - 1x_3 = 4 \\
0x_1 - 1x_2 + 2x_3 = 3
\end{cases}
$$

写成矩阵形式 $A \mathbf{x} = B$：
$$
\begin{bmatrix}
2 & -1 & 0 \\
-1 & 3 & -1 \\
0 & -1 & 2
\end{bmatrix}
\begin{bmatrix}
x_1 \\ x_2 \\ x_3
\end{bmatrix}
=
\begin{bmatrix}
3 \\ 4 \\ 3
\end{bmatrix}
$$

**为什么叫三对角矩阵？**
你看系数矩阵 $A$，非零元素只集中在**主对角线**以及它的**紧邻的上方和下方**。
- $(1,3)$ 位置是0，$(3,1)$ 位置是0。
- 这意味着：计算第 $j$ 列的值，只和 $j-1$（左边）以及 $j+1$（右边）有关。这就是物理意义上的“只和左右邻居相关”。

---

### 第四步：求解第2行（高斯消元/代入法）

我们要解出 $x_1, x_2, x_3$。

**1. 利用方程1消去 $x_1$：**
由 $2x_1 - x_2 = 3 \Rightarrow x_1 = 1.5 + 0.5x_2$。
这就建立了一个关系：**左边的数可以用右边的数表示**。

**2. 代入方程2：**
$-1(1.5 + 0.5x_2) + 3x_2 - x_3 = 4$
$-1.5 - 0.5x_2 + 3x_2 - x_3 = 4$
$2.5x_2 - x_3 = 5.5$
$\Rightarrow x_2 = \frac{5.5 + x_3}{2.5} = 2.2 + 0.4x_3$
你看，现在 $x_2$ 也被表示成了 $x_3$ 的函数。

**3. 代入方程3：**
$-x_2 + 2x_3 = 3$
$-(2.2 + 0.4x_3) + 2x_3 = 3$
$-2.2 - 0.4x_3 + 2x_3 = 3$
$1.6x_3 = 5.2$
**$x_3 = 3.25$**

**4. 回代求出其他值：**
$x_2 = 2.2 + 0.4 \times 3.25 = 2.2 + 1.3 = \mathbf{3.5}$
$x_1 = 1.5 + 0.5 \times 3.5 = 1.5 + 1.75 = \mathbf{3.25}$

所以第2行的期望值为：
$$ E_{2,1} = 3.25, \quad E_{2,2} = 3.5, \quad E_{2,3} = 3.25 $$

---

### 第五步：求解第1行（同样的方法）

现在我们需要求目标行 $i=1$。
设 $y_1 = E_{1,1}, \quad y_2 = E_{1,2}, \quad y_3 = E_{1,3}$。
这一步的关键是：**方程右边的常数项变了**，因为下一行（第2行）的期望值不再是0，而是刚才算出来的 3.25, 3.5, 3.25。

**1. 位置 (1, 1) 方程：**
$$ y_1 = 1 + \frac{1}{3}y_1 + \frac{1}{3}y_2 + \frac{1}{3}E_{2,1} $$
代入 $E_{2,1}=3.25$：
$$ y_1 = 1 + \frac{1}{3}y_1 + \frac{1}{3}y_2 + \frac{1}{3}(3.25) $$
$$ \frac{2}{3}y_1 - \frac{1}{3}y_2 = 1 + 1.0833... $$
整理得：$2y_1 - y_2 = 3 + 3.25 = 6.25$

**2. 位置 (1, 2) 方程：**
$$ y_2 = 1 + \frac{1}{4}y_2 + \frac{1}{4}y_1 + \frac{1}{4}y_3 + \frac{1}{4}E_{2,2} $$
代入 $E_{2,2}=3.5$：
$$ \frac{3}{4}y_2 - \frac{1}{4}y_1 - \frac{1}{4}y_3 = 1 + \frac{3.5}{4} $$
整理得：$-y_1 + 3y_2 - y_3 = 4 + 3.5 = 7.5$

**3. 位置 (1, 3) 方程：**
$$ y_3 = 1 + \frac{1}{3}y_3 + \frac{1}{3}y_2 + \frac{1}{3}E_{2,3} $$
代入 $E_{2,3}=3.25$：
$$ \frac{2}{3}y_3 - \frac{1}{3}y_2 = 1 + \frac{3.25}{3} $$
整理得：$-y_2 + 2y_3 = 3 + 3.25 = 6.25$

**解这个新的方程组：**
1. $2y_1 - y_2 = 6.25 \Rightarrow y_1 = 3.125 + 0.5y_2$
2. 代入中间方程：
   $-(3.125 + 0.5y_2) + 3y_2 - y_3 = 7.5$
   $2.5y_2 - y_3 = 10.625 \Rightarrow y_2 = 4.25 + 0.4y_3$
3. 代入最后方程：
   $-(4.25 + 0.4y_3) + 2y_3 = 6.25$
   $1.6y_3 = 10.5$
   **$y_3 = 6.5625$**

回代：
$y_2 = 4.25 + 0.4 \times 6.5625 = 4.25 + 2.625 = \mathbf{6.875}$

题目问的是 $(1, 2)$ 出发，即 $y_2$。
所以最终答案是 **6.87500**。

### 总结
1. **转化思路**：把每一行看作一个整体。因为同一行的 $E(i, j)$ 互相依赖，所以必须联立方程解。
2. **三对角矩阵**：因为网格图中，每个格子只和左右邻居发生关系，所以方程系数只在 $j-1, j, j+1$ 处非零，形成了三对角。
3. **求解技巧**：从左到右把 $x_j$ 表示为 $A \cdot x_{j+1} + B$ 的形式，最后算出最右边的值，再倒推回来。这就是这一类 DP 优化的核心（实际上就是简化版的高斯消元）。

求解三对角矩阵的经典算法叫作 **追赶法（Thomas Algorithm）**。

它本质上就是高斯消元法的简化版��因为矩阵大部分位置都是0，我们不需要遍历那些0，从而把时间复杂度从 $O(N^3)$ 降低到了 **$O(N)$**。

以下是详细的编程思路和推导。

---

### 1. 什么是三对角矩阵方程？

在你的题目中，对于每一行（固定 $i$），我们都有一组关于 $j$ （从 1 到 $m$）的方程。形式如下：

$$ a_j x_{j-1} + b_j x_j + c_j x_{j+1} = d_j $$

其中：
*   $x_j$ 是我们想求的未知数（即 $dp[i][j]$）。
*   $a_j, b_j, c_j$ 是系数（由概率决定）。
*   $d_j$ 是常数项（包含 $+1$ 和下一行 $dp[i+1][j]$ 的贡献）。

**注意边界：**
*   当 $j=1$ 时，$x_0$ 不存在，所以 $a_1 = 0$。
*   当 $j=m$ 时，$x_{m+1}$ 不存在，所以 $c_m = 0$。

矩阵长这样：
$$
\begin{bmatrix}
b_1 & c_1 & 0 & \dots & 0 \\
a_2 & b_2 & c_2 & \dots & 0 \\
0 & a_3 & b_3 & c_3 & \dots \\
\vdots & \vdots & \vdots & \ddots & \vdots \\
0 & 0 & \dots & a_m & b_m
\end{bmatrix}
\begin{bmatrix}
x_1 \\
x_2 \\
x_3 \\
\vdots \\
x_m
\end{bmatrix}
=
\begin{bmatrix}
d_1 \\
d_2 \\
d_3 \\
\vdots \\
d_m
\end{bmatrix}
$$

---

### 2. 核心思路：消元（“追”）与 回代（“赶”）

我们希望把 $x_j$ 表示成只和后一项 $x_{j+1}$ 有关的形式：
$$ x_j = C'_j \cdot x_{j+1} + D'_j $$
这里的 $C'_j$ 和 $D'_j$ 是我们要一边遍历一边算出来的中间变量。

#### 第一步：推导递推公式
将 $x_{j-1} = C'_{j-1} x_j + D'_{j-1}$ 代入原方程 $a_j x_{j-1} + b_j x_j + c_j x_{j+1} = d_j$ 中：

1.  **代入前一项**：
    $$ a_j (C'_{j-1} x_j + D'_{j-1}) + b_j x_j + c_j x_{j+1} = d_j $$

2.  **整理 $x_j$ 的系数**：
    $$ (a_j C'_{j-1} + b_j) x_j + c_j x_{j+1} = d_j - a_j D'_{j-1} $$

3.  **移项求 $x_j$**：
    $$ x_j = \frac{-c_j}{a_j C'_{j-1} + b_j} x_{j+1} + \frac{d_j - a_j D'_{j-1}}{a_j C'_{j-1} + b_j} $$

4.  **对比目标形式 $x_j = C'_j x_{j+1} + D'_j$**，我们得到递推式：
    $$ C'_j = \frac{-c_j}{a_j C'_{j-1} + b_j} $$
    $$ D'_j = \frac{d_j - a_j D'_{j-1}}{a_j C'_{j-1} + b_j} $$

**初始化（$j=1$ 时）**：
因为 $a_1 = 0$，所以：
$$ C'_1 = \frac{-c_1}{b_1}, \quad D'_1 = \frac{d_1}{b_1} $$

---

### 3. 编程实现步骤

#### 步骤一：预处理每一行的系数 (a, b, c, d)
你需要根据题目逻辑（在角落、在边缘、在中间）填充 $a, b, c, d$ 数组。
*   $a[j]$：来自 $dp[i][j-1]$ 的系数（左边）
*   $b[j]$：来自 $dp[i][j]$ 的系数（自己）
*   $c[j]$：来自 $dp[i][j+1]$ 的系数（右边）
*   $d[j]$：常数项 $1 + P_{down} \times dp[i+1][j]$

**一定要注意移项变号！**
例如方程是 $dp[i][j] = 1 + p \cdot dp[i][j] + \dots$
移项后变成 $(1-p)dp[i][j] - \dots = 1 + \dots$
所以 $b[j]$ 通常是正数，$a[j], c[j]$ 是负数。

#### 步骤二：正向扫描（计算 C' 和 D'）
从 $j=1$ 到 $m-1$ 计算中间系数。

```cpp
// cp[j] 对应上面的 C'_j, dp_temp[j] 对应 D'_j
// temp 是分母 common part
double temp = b[1];
cp[1] = -c[1] / temp;
dp_temp[1] = d[1] / temp;

for (int j = 2; j <= m; j++) {
    temp = b[j] + a[j] * cp[j-1]; 
    cp[j] = -c[j] / temp;
    dp_temp[j] = (d[j] - a[j] * dp_temp[j-1]) / temp;
}
```

#### 步骤三：反向回代（计算 x）
求出 $x_m$ 后，倒着推回 $x_1$。
注意：$x_m$ 的方程中 $c_m=0$，所以 $x_m = D'_m$（实际上在上面的循环里算出来的 $cp[m]$ 会是 0）。

```cpp
// 初始化结果数组 x
// 最后一个元素的值就是 dp_temp[m]
x[m] = dp_temp[m];

for (int j = m - 1; j >= 1; j--) {
    x[j] = cp[j] * x[j+1] + dp_temp[j];
}
```

### 4. 完整的 C++ 代码模板

结合你的题目背景，这里是一个可以直接参考的函数模板：

```cpp
#include <iostream>
#include <vector>
#include <iomanip>

using namespace std;

// 用于存储求解结果的 DP 表
// dp[i][j] 表示 (i, j) 到第 n 行的期望步数
double dp[2005][2005]; 

// 三对角矩阵求解函数
// m: 列数
// a, b, c: 系数数组 (注意下标从1开始)
// d: 常数数组
// x: 结果数组 (也就是 dp[i] 这一行)
void solve_tridiagonal(int m, vector<double>& a, vector<double>& b, vector<double>& c, vector<double>& d, double* x) {
    // cp 和 dp_temp 用于存储中间推导过程的系数 C' 和 D'
    vector<double> cp(m + 1);
    vector<double> dp_temp(m + 1);

    // 1. 初始化第一项
    // 分母通常不会为0，但在严谨的数学库中需要检查
    double denom = b[1]; 
    cp[1] = -c[1] / denom;
    dp_temp[1] = d[1] / denom;

    // 2. 正向扫描 (从第2个到第m个)
    for (int j = 2; j <= m; j++) {
        denom = b[j] + a[j] * cp[j - 1];
        cp[j] = -c[j] / denom;
        dp_temp[j] = (d[j] - a[j] * dp_temp[j - 1]) / denom;
    }

    // 3. 反向回代求 x
    x[m] = dp_temp[m];
    for (int j = m - 1; j >= 1; j--) {
        x[j] = cp[j] * x[j + 1] + dp_temp[j];
    }
}

int main() {
    int n, m, start_x, start_y;
    cin >> n >> m >> start_x >> start_y;

    // 边界条件：第 n 行期望步数为 0
    for (int j = 1; j <= m; j++) dp[n][j] = 0;

    // 只有1列的特殊情况直接处理
    if (m == 1) {
        printf("%.5f\n", (double)(n - start_x) * 2.0);
        return 0;
    }

    // 从 n-1 行倒着推到 start_x 行
    for (int i = n - 1; i >= start_x; i--) {
        vector<double> A(m + 1), B(m + 1), C(m + 1), D(m + 1);

        for (int j = 1; j <= m; j++) {
            // cnt: 可选方向数
            int cnt = 0;
            // 统计可行方向：
            // 必定可以：原地(1) + 向下(1) = 2
            cnt += 2; 
            if (j > 1) cnt++; // 可以向左
            if (j < m) cnt++; // 可以向右

            // 原始方程：
            // x[j] = 1 + (1/cnt)*x[j] + (1/cnt)*x[j-1] + (1/cnt)*x[j+1] + (1/cnt)*dp[i+1][j]
            
            // 移项整理为 A*x[j-1] + B*x[j] + C*x[j+1] = D
            
            // x[j-1] 的系数 (向左走)
            if (j > 1) A[j] = -1.0 / cnt; 
            else A[j] = 0;

            // x[j+1] 的系数 (向右走)
            if (j < m) C[j] = -1.0 / cnt;
            else C[j] = 0;

            // x[j] 的系数 (原地不动 + 移项过来的1)
            // 原方程右边有 (1/cnt)*x[j]，移到左边变成 x[j] - (1/cnt)*x[j]
            B[j] = 1.0 - (1.0 / cnt);

            // 常数项 D
            // 包含 1 (走一步的时间) 和 下一行的期望值 (向南走)
            D[j] = 1.0 + (1.0 / cnt) * dp[i + 1][j];
        }

        // 解这一行的方程组，结果直接存入 dp[i]
        solve_tridiagonal(m, A, B, C, D, dp[i]);
    }

    printf("%.5f\n", dp[start_x][start_y]);

    return 0;
}
```
