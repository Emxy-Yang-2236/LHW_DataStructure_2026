
### 第一步：理解“01串的最长不下降子序列”
在一个只有 0 和 1 的字符串中，一个“不下降子序列”长什么样？
它一定是由**前面一段 0 和后面一段 1** 组成的（当然也可以全是一段 0，或者全是一段 1）。
形如：`0 0 0 ... 1 1 1`

这就意味着，对于给定的查询区间 `[l, r]`，一定存在一个**分界点 $k$**（其中 $l-1 \le k \le r$）：
- 在区间 `[l, k]` 中，我们只取 `0`；
- 在区间 `[k+1, r]` 中，我们只取 `1`。

> 注意：$k=l-1$ 表示全取1（没有0）；$k=r$ 表示全取0（没有1）。

---

### 第二步：数学公式推导（核心逻辑）
假设我们已经维护了前缀和：
- $cnt0[i]$ 表示前 $i$ 个字符中有多少个 `0`。
- $cnt1[i]$ 表示前 $i$ 个字符中有多少个 `1`。

那么，对于某个特定的分界点 $k$，我们在这个区间能得到的最长不下降子序列长度是：
**长度 $L(k)$ = (区��� $[l, k]$ 中 0 的个数) + (区间 $[k+1, r]$ 中 1 的个数)**

代入前缀和公式：
$$L(k) = (cnt0[k] - cnt0[l-1]) + (cnt1[r] - cnt1[k])$$

我们把这个式子**重新组合**一下，把和 $k$ 相关的项放在一起，和 $l, r$ 相关的项放在一起：
$$L(k) = (cnt0[k] - cnt1[k]) - cnt0[l-1] + cnt1[r]$$

---

### 第三步：为什么用 ST 表？
观察上面的最终公式：
对于每一次询问 `[l, r]` 来说，**$cnt0[l-1]$ 和 $cnt1[r]$ 都是固定不变的常数**。
我们要想让 $L(k)$ 最大，就只需要让 **$(cnt0[k] - cnt1[k])$ 最大** 即可！

所以，解题思路呼之欲出：
1. 我们构造一个新的数组 $V$，令 **$V[i] = cnt0[i] - cnt1[i]$**。
2. 题目变成了：在每次查询时，求区间 **$[l-1, r]$ 内 $V$ 数组的最大值**。
3. 求出最大值后，加上 $cnt1[r] - cnt0[l-1]$，就是单次询问的答案。

这是一个标准的 **静态区间最值查询 (RMQ)** 问题！
- 数组 $V$ 是静态的（不修改）。
- 询问次数 $m$ 极大（达到 $5 \times 10^6$）。
- 用 ST 表可以在 $O(N \log N)$ 预处理后，实现 **$O(1)$ 极速查询**。
